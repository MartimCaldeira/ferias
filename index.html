Credibilidade das agências

Abreu

Fundada em 1840. Nº1 de reputação em Portugal. +120 lojas físicas. Ideal para grupos.

Top Atlântico

Nº3 de reputação. Agência portuguesa, boa para pacotes de praia.

Logitravel

Online, líder PT/ES. Boa para comparar preços — mas confirma se a bagagem está incluída.

Google Flights

Só para comparar — reserva sempre na companhia aérea ou agência conhecida.

Recomendação final para 7 pessoas

Para grupos grandes, o ideal é ligar diretamente à Abreu e pedir orçamento de grupo — conseguem negociar tarifas especiais para 7 pax que não aparecem online. Maiorca via Sevilha é a opção mais barata. Creta é o melhor custo-benefício se quiserem algo mais autêntico.
